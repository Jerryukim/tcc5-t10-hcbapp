WEB3 HEALTHCARE BOOKING APP

OVERVIEW
 Web3 Healthcare Booking App is a decentralized healthcare appointment system built for the TechCrush Web3 Bootcamp Cohort 5 Capstone Project.
The platform demonstrates how blockchain technology can be integrated with traditional web applications to improve transparency, security, and trust in healthcare systems.
Patients can book appointments and pay using MetaMask, while doctors can create medical records whose hashes are stored on-chain for verification.

KEY FEATURES
*Doctor appointment booking system
*Web3 payments using MetaMask
*Smart contract interaction on Sepolia testnet
*Medical record hash verification on blockchain
*Patient and doctor portal system
*Backend API for managing bookings and records

TECH STACK
FRONTEND: Next.js, React, TypeScript, TailwindCSS, 
BACKEND: Node.js, Prisma ORM, PostgresQL (Neon) 
BLOCKCHAIN: Foundry, Ether.js, Sepolia Testnet, Wallet, MetaMask, Smart Contract


Network: Sepolia Testnet
Contract Address:0x67a54D7D9B17127Bc91508839915A8285245cB16

The smart contract handles:
*Appointment payments
*Storage of medical record hashes
*Verification of stored hashes

APPLICATION FLOW
Patient Flow:
Select medical specialty
Choose a doctor
Book appointment
Pay using MetaMask

Doctor Flow:
Accept patient booking
Record diagnosis and prescription
Generate record hash
Store hash on blockchain

RUNNING THE PROJECT LOCALLY
Clone the repository: git clone https://github.com/JerryUkim/tcc5-t10-hcbapp.git
Navigate to the project folder: cd tcc5-t10-hcbapp
Install dependencies: npm install
Run the development server: npm run dev
Open in browser: http://localhost:3000
